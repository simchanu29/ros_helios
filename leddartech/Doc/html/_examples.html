Leddar™ SDK sample code

Some examples using the SDK are installed with the Leddar™ SDK. Under Windows they are installed in a LeddarTech folder in your Documents folder. If you intend to modify the examples, it is recommended that you copy them to another location, otherwise they will be overwritten if you install a newer version of the SDK.

Under Linux the examples are in the SDK directory.

Examples are provided for:

  • LeddarC Example
  • LeddarNET Example
  • MATLAB™ Integration Example
  • LabVIEW™ Integration Example
  • RS-485 Communication Demo

All example code is commented and should be easy to understand.

LeddarC Example

Under Windows this example is in the Examples solution with the LeddarNET example. This is a Visual Studio 2008 solution. If you are using a more recent version, Visual Studio should convert it without any issues. Before executing the example, select the proper platform for your installation (Win32 or x64).

Under Linux a Makefile is provided to rebuild the demo but a pre-built binary is also provided in the main Release directory.

The LeddarC example is a console application driven by simple text menus. This was done so that usage of the LeddarC interface is not obscured by complex and lengthy GUI code.

LeddarNET Example

This example is in the Examples.sln solution with the LeddarC example. This is a Visual Studio 2008 solution. If you are using a more recent version, Visual Studio should convert it without any issues.

The LeddarNET example is a simple Windows Forms application. This application is neither very user-friendly nor powerful. Its goal is to illustrate the usage of all LeddarNET features.

Before executing the example, select the proper platform for your installation (Win32 or x64).
