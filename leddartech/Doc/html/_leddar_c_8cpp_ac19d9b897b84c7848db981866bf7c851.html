unsigned int  aId,
unsigned int  aIndex,

Generic result reader. This function should only be called while in the "new data" callback function.

Parameters
aHandleHandle returned by LeddarCreate.
aIdId of the result to read as defined in LeddarResults.h
aIndexIndex for array results. Set to 0 for single results.
aValuePointer to a variable that will receive the value.
Returns
Standard LeddarC return code.
