Welcome to the Leddar™ SDK!

The Leddar™ SDK is a set of interfaces to allow programmers to integrate Leddar™ sensors in their products. Interfaces are provided for C and .NET which should allow easy integration with most environments.

The C interface allows low level integration in C, C++ or any other languages that easily interface with C functions.

The .NET interface allows easy usage in .NET languages like C#, Visual Basic but also in environments that support .NET integration like MATLAB™ or LabVIEW™.

To help developers get started quickly with the SDK, sample code consisting of example applications in Visual Studio and integration examples in MATLAB™ and LabVIEW™ are provided.

The goal of this documentation is not to explain Leddar™ sensors. It is assumed that you have read the sensor documentation and that you understand basic Leddar™ concepts.

Note
The Leddar™ SDK is available for Windows and Linux. However the .NET interface as well as the MATLAB™ and LabVIEW™ examples are only available for the Windows platform.
