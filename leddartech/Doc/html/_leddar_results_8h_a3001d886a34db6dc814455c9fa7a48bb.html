RID_DETECTION_SEGMENT 

The segment for the detection at the given index.

RID_DETECTION_DISTANCE 

The distance in meters for the detection at the given index.

RID_TEMPERATURE 

The device temperature in Celsius.

RID_LED_INTENSITY 

The current LED intensity.

If manual intensity mode is configured, this will be the same as PID_LED_INTENSITY. If automatic mode is configured this will give the intensity computed by the sensor.

RID_GAIN_ENABLED 

Reserved for internal use.

RID_OUTPUT_STATE 

The current state of the output at the given index.

This will be 1 if there is a detection, 0 if there is no detection.

Note
Although this result is supported by all models, its value is currently only meaningful for IS16 industrial Leddar™ sensor.
