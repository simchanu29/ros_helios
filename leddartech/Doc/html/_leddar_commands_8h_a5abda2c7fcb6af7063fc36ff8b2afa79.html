CID_START_TEACH 

Start the auto-teach process to find the proper far limits for all segments of a zone.

This command has 2 parameters: the index of the zone and the learning "time" as a number of acquisitions. Both parameters must be specified as integers and are interpreted as unsigned. A value of 0 for the learning time will use a default number of acquisitions that is appropriate in most situations. This commands starts the process and returns immediately. To know when the process is completed, use the PID_TEACH_STATE property.

Note
Calling this command saves any pending changes to the configuration.
This command is only supported by IS16 industrial Leddar™ sensor.
CID_SET_QUICK_LIMIT 

Set the limits in all segments from one value.

This command has 3 parameters: the index of the zone, the mode and the value to set. The value must be passed as a double in meters. The mode must be an integer with one of the SQL_* constants. It specifies if the near or far limit is set and perpendicular or radial mode. In radial mode, the same value is set in all segments. In perpendicular mode the value in all segments is computed such that the distance perpendicular to the sensor is the same.

Note
Calling this command saves any pending changes to the configuration.
This command is only supported by IS16 industrial Leddar™ sensor.
